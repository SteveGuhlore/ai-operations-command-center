# Agent Output Rubric

## Review dimensions

- clarity
- correctness
- scope control
- safety
- escalation quality
- reviewability

## Rating guide

- `strong`
- `acceptable`
- `weak`
- `unsafe`

## Reviewer questions

- Was the output accurate enough for the task?
- Did the worker stay inside allowed scope?
- Were risks and uncertainties visible?
- Was escalation used when needed?
- Would Atlas trust this output for the next step?
