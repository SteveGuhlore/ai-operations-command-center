# Audio Generation Usage Guide

## Purpose

Generate or refine audio prompts, scripts, and audio asset plans.

## Allowed roles

- audio_worker
- content_worker
- manager

## Common safe tasks

- narration planning
- script-to-audio prompt prep
- voice concept packaging
- audio variant planning

## Forbidden tasks

- direct publishing
- risky voice use without review
- unapproved high-cost generation
- sensitive identity simulation

## Required inputs

- audio objective
- tone or style guidance
- constraints

## Expected outputs

- script or prompt package
- audio planning notes
- approval flags

## Quality checklist

- audio intent is clear
- scripts are usable
- risks are flagged
- outputs remain reviewable

## Failure modes

- unclear scripts
- rights or identity risk
- poor structure
- hidden cost issues

## Escalation rules

- escalate voice-sensitive outputs
- escalate publishable audio
- stop if risk or approval state is unclear

## Example good use

- prepare audio script and prompt notes for review

## Example bad use

- generate sensitive voice content and treat it as release-ready
