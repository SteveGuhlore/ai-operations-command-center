# Web Research Usage Guide

## Purpose

Gather current external information when browsing is explicitly allowed.

## Allowed roles

- manager
- debug_worker
- content_worker
- guard_worker
- budget_worker

## Common safe tasks

- verify current facts
- collect current comparison notes
- check current documentation
- gather approval-relevant context

## Forbidden tasks

- browsing without permission when it is required
- collecting secrets or private account data
- posting externally
- acting on results without review when approval is required

## Required inputs

- clear question
- allowed browsing scope
- reason current information is needed

## Expected outputs

- sourced summary
- relevant facts
- uncertainty notes

## Quality checklist

- current sources are used when needed
- results are attributable
- risky assumptions are avoided
- output is concise and useful

## Failure modes

- outdated or weak sources
- unsupported claims
- scope creep into unrelated browsing
- misuse of external information

## Escalation rules

- escalate if the query affects spending, compliance, or safety
- stop if browsing would violate permissions
- escalate if source quality is unclear

## Example good use

- check current documentation before writing guidance

## Example bad use

- guess at current facts instead of verifying them
