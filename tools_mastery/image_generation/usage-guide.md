# Image Generation Usage Guide

## Purpose

Create or refine still-image concepts and visual assets.

## Allowed roles

- media_worker
- content_worker
- manager

## Common safe tasks

- image prompt drafting
- concept variants
- asset packaging notes
- visual planning

## Forbidden tasks

- direct publishing
- brand release without approval
- unsafe or noncompliant visual generation
- unapproved high-cost generation

## Required inputs

- visual objective
- audience or use context
- constraints

## Expected outputs

- prompt set
- asset concept summary
- moderation or approval notes

## Quality checklist

- prompts are clear
- visuals match objective
- moderation concerns are flagged
- cost-sensitive usage is acknowledged

## Failure modes

- vague prompts
- unusable assets
- ignored moderation concerns
- overspend risk

## Escalation rules

- escalate publishable assets
- escalate moderation concerns
- escalate large generation requests

## Example good use

- prepare a structured prompt pack for review

## Example bad use

- generate and release brand-facing images without approval
