# Video Generation Usage Guide

## Purpose

Generate or refine video concepts, prompts, and video asset plans.

## Allowed roles

- media_worker
- manager

## Common safe tasks

- short-form video concepting
- prompt drafting
- shot sequence planning
- batch concept preparation

## Forbidden tasks

- direct publishing
- risky brand release
- unapproved high-cost generation
- skipping moderation review

## Required inputs

- content objective
- format target
- style or audience guidance

## Expected outputs

- video concept summary
- prompt set
- shot or structure notes
- approval flags

## Quality checklist

- concept is clear
- output is reviewable
- moderation concerns are visible
- cost posture is acknowledged

## Failure modes

- vague video direction
- unusable prompts
- ignored moderation concerns
- overspend risk

## Escalation rules

- escalate publishable assets
- escalate high-cost generation
- stop if release pressure bypasses approval

## Example good use

- prepare a short-form concept batch for review

## Example bad use

- treat generated video as ready to publish without checks
