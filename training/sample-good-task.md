# Sample Good Task

## Why it is good

- scope is small
- files are bounded
- stop conditions are explicit
- outputs are reviewable

## Example

```text
Goal: Update a single validator message for clarity.
Allowed files: scripts/validate-tasks.ps1
Forbidden files: config/**, .env
Acceptance criteria: message is clearer, behavior unchanged
Stop conditions: stop if more than one script must change
```
