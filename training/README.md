# Training

## Purpose

This folder contains agent conditioning and playbook training materials for the AI Operations Command Center.

## Important note

This is not model fine-tuning.

It is operational conditioning, memory preparation, evaluation preparation, and role playbook training.

It is:

- role guidance
- behavioral conditioning
- quality standards
- escalation standards
- formatting standards
- examples of good and bad task and output patterns

## Contents

- role playbooks for core agents
- revenue production playbooks (`maker-digital-product-playbook.md`, `market-marketing-worker-playbook.md`)
- optional specialist playbook
- evaluation rubric
- sample good task
- sample bad task
- sample worker output

## Revenue production playbooks

- [maker-digital-product-playbook.md](./maker-digital-product-playbook.md) — `Maker` / `digital_product_worker`
- [market-marketing-worker-playbook.md](./market-marketing-worker-playbook.md) — `Market` / `marketing_worker` (central marketer for all agents and revenue pods)

## Usage

These files are intended to guide:

- prompt design
- task shaping
- output review
- worker evaluation
- future memory and playbook systems

It is not:

- frontier-model training
- provider integration
- autonomous runtime behavior
