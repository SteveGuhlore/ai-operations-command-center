# Shortcuts

## Purpose

These local shortcut helpers are placeholders for opening the Star Office UI dashboard on a machine where the UI project exists separately.

## Usage

1. Run [Start-Star-Office-UI.bat](C:\Users\sbattaglia\Downloads\AI Operations Command Center\shortcuts\Start-Star-Office-UI.bat) first.
2. Wait for the Star Office UI dev server to start.
3. Then open [Open-Star-Office-Dashboard.url](C:\Users\sbattaglia\Downloads\AI Operations Command Center\shortcuts\Open-Star-Office-Dashboard.url).

## Important note

The dashboard URL only works after the dev server is already running.

## Placeholder values

- Star Office UI folder path: `C:\Users\sbattaglia\Downloads\Star-Office-UI`
- Dashboard URL: `http://localhost:3000`
