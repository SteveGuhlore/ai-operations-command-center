# Batch Review Checklist

Codex Manager checks:

- Did each worker touch only allowed files?
- Did any worker touch forbidden files?
- Did tests/checks run?
- Were failures logged?
- Are docs/handoffs updated?
- Is live trading still disabled?
- Are API keys/secrets absent?
- Are tasks small enough to review?
- Should failed tasks be requeued, escalated, or abandoned?
