# Autonomy Levels

Level 0: Planning only. No edits.

Level 1: Task creation and dry-run worker loops. Human starts commands.

Level 2: Workers run safe tasks. Human approves merge.

Level 3: Safe docs/tests/simple fixes may auto-complete if checks pass.

Level 4: Hours-long safe-tier batches with budget, lock, report, and forbidden-file guards.

This command center foundation starts at **Level 1**.

Move to Level 2 only after:
- doctor and validation checks pass,
- dry-run behavior is understood,
- a real target project is explicitly reviewed.

Move to Level 3 only after several clean batches.
